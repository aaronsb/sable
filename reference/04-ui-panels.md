# UI Panels and Controls

## Left Panel — Layer Controls
- Layer toggles (48 checkboxes grouped by category: aviation, maritime, space, geophysical, geopolitical, infrastructure)
- Entity count display per category
- GIBS imagery controls (date slider, opacity, play animation)
- Theme toggle (light/dark)
- HUD color cycle (7 accent colors)
- Legend button
- Settings button
- Minimize/maximize

## Right Panel — Entity Detail
- Selected entity detail card (name, type, position, speed, heading, altitude)
- Wikipedia image (fetched on-demand)
- Metrics table (speed knots, heading degrees, altitude feet, etc.)
- Action buttons: fly to, calculate route, eavesdrop, region dossier
- Media player (HLS video for CCTV, audio for radio intercepts)

## News Feed Panel
- Scrollable geolocated news articles
- Risk score color coding (red=high, orange=medium)
- Source attribution + relative publish time
- Click article → fly to location on map
- Inline wiki images for multi-source articles

## Radio Intercept Panel
- Top 50 Broadcastify feeds by listener count
- OpenMHz system search (nearest by location)
- Audio player with volume control
- Frequency + talkgroup metadata
- Scan mode (auto-cycle nearby systems)
- Eavesdrop mode toggle (map click-to-intercept)

## Filter Panel
- Advanced filter modal (country, airline, ship type, etc.)
- Flight category toggles (commercial, military, private, tracked)
- Ship category toggles (military, cargo, passenger, yachts)
- Text search (callsign, ship name, registration)

## Settings Modal
- API key registry (add/edit/delete keys for 20+ services)
- News feed configuration (up to 20 RSS feeds with weight 1-5)
- Admin key input
- Reset to defaults

## Top Bar
- Find/locate bar (search by callsign, ship name, coordinates)
- Scale bar
- Changelog modal
- Onboarding modal (keyboard shortcuts, feature tour)
